# Objetivos do Projeto

## 1) Bot Premium (Receita recorrente)
- Automatizar comandos úteis no chat.
- Reduzir carga operacional do streamer/moderação.
- Aumentar percepção de valor para assinatura Pro.

## 2) Clipagem Automática (Aquisição)
- Detectar momentos de alta energia da live.
- Sugerir highlights para TikTok/Shorts.
- Gerar conteúdo de topo de funil para crescimento orgânico.

## 3) CRM de Comunidade (Retenção)
- Pontuar e segmentar audiência por comportamento.
- Criar campanhas direcionadas para retorno de viewers.
- Melhorar retenção e LTV dos creators.

## 4) Safety de Comunidade (Confiança)
- Detectar sinais de comportamento ofensivo recorrente entre lives.
- Apoiar moderadores com alertas e contexto de risco.
- Reduzir reincidência de abuso no chat sem automatizar banimentos cegos.
